# 詞序 VocabFlow

以大考中心現行高中英文參考詞彙表為基礎的每日學習網站，共收錄 6,004 個官方詞條。

## 網站

https://zozo971209-pixel.github.io/vocabflow-6004/

## 功能

- 前 120 天每天固定 50 詞，第 121 天為最後 4 詞
- 每天平均混合第 1–6 級與不同字母開頭
- 英文與中文朗讀，可切換慢速／正常
- 已熟悉、待複習與不熟標記
- 搜尋、級別與狀態篩選
- 學習進度保存在使用者自己的瀏覽器
- 手機與桌面響應式介面

## 本機執行

```bash
npm install
npm run dev
```

## 資料說明

英文詞彙、官方詞性與六級分級來自大學入學考試中心《高中英文參考詞彙表（111學年度起適用）》。中文釋義與音標由 ECDICT 資料補充，並非大考中心官方中文翻譯。
